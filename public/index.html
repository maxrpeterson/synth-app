About this project

A three-oscillator synthesizer you can play with your keyboard. The keys in the "asdf..." row control the white keys, and the black keys are in the "qwer..." row. To control the knobs, click and drag your mouse ↑ and ↓.

I created this project for my final project for General Assembly's Web Development Immersive class. I used Tone.js to help simplify the synth code, and Qwerty Hancock for the keyboard. There is a very small Node.js & Express.js server on the back end, which saves presets to a MongoDB collection.

Check out my GitHub and LinkedIn!
